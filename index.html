Bienvenido a este Layout!

Este Layout es una pequeña practica siguiendo las instrucciones del desarollador Falcon Masters, sobre como crear un Layout en alguno de sus cursos. La estructura semantica se mantiene casi igual, sin embargo, hay elementos que no poseen el mismo diseño. Me gusta la tematica vaporwave. Por ello, he decidido diseñar este Layout con tales colores. He usado node.js también para llamar algunas dependencias, como normalize.css
